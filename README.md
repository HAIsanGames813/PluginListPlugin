# プラグイン管理リストプラグイン

## ダウンロード
[ここからダウンロード](https://github.com/HAIsanGames813/PluginListPlugin/releases)しダウンロードしたファイルをクリック

## 概要
プラグインを管理できるツールウィンドウを追加するプラグインです

## 使用方法
画面左上にあるツールから「プラグイン管理リスト」を選択肢使用できます  
プラグイン名横のボタンをクリック、  
またはシフトまたはコントロールで一括選択し無効化/有効化、削除ができます
